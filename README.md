# Parixit Sanghani Portfolio

This repository contains information about Parixit Sanghani.
It is a React + TypeScript portfolio codebase that presents profile details, experience, skills, and projects.

## Codebase Overview

The app includes:
- Header and top-level layout
- Professional summary
- Work experience
- Technical skills
- Education
- Interests
- Projects

## Tech Stack

- React
- TypeScript
- CSS
- Create React App (`react-scripts`)

## Project Structure

```text
src/
  components/
    header/
    profile/
    summary/
    experience/
    technicalSkills/
    education/
    interests/
    projects/
  static/
```

## Local Development

1. Install dependencies: `npm install`
2. Start development server: `npm start`
3. Open `http://localhost:3000`

## Useful Scripts

- `npm start` - Run the app locally
- `npm test` - Run tests
- `npm run build` - Build for production
